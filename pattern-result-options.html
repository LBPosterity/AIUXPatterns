Result Options

Overview

Providing option alongside each result allows users to easily utilise them. The options vary between contexts, however, common utilities like copy text are obvious to include.

User Archetype

All users want to quickly action their results so common actions like copy help all users. Other functions like auto play can be highly convenient for anyone but also make products more accessible.

Considerations

Results options can also serve as important feedback mechanisms, the thumb up & down can allows users to feedback on individual results which products can correlate to the prompt & analyse.

A counter intuitive option to include is "regenerate", however, because LLM's can provide different results for the same prompt, a unique AI interaction is to regenerate the same prompt multiple times to compare & contrast results to pick the best.
